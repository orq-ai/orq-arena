02Live-fire findings — what the real router taught us

Four smoke runs against api.orq.ai/v3/router did more than verify wiring; they produced the report's best material.

LIVE 01 · the predicted confound, observed, then eliminated

kimi-k2.6 finished at ELO −3000 — because of a vendor default, not quality

In the first round-robin smoke, kimi-k2.6 thought by default, burned 358–399 of its 400-token budget on reasoning every single round (finish_reason=length), answered in stumps, and lost every round. Exactly the "ranking conflates model quality with vendor defaults" failure the plan predicted — measured in production. A live audit of the whole roster followed: gemini-3-pro-preview is broken upstream (router 500), and kimi-k2.6, deepseek-v4-pro and qwen3.5-flash cannot have thinking disabled via the router. The default pool now contains only models verified thinking-OFF (gemini-3.1-pro-preview + disable = 0 reasoning tokens; deepseek-chat; mistral-medium-2604); the un-disableable thinkers moved to configs/reasoning_arena.yaml where always-on is the uniform policy.

LIVE 02 · the methodology paying rent

Cheap judges flip 67–83% under seat-swapping — and the ranking survives anyway

On easy prompts the panel's coin flips became visible: in the very first smoke, two of three judges contradicted themselves when A and B swapped seats, abstained, and the round resolved inconclusive instead of minting a fake verdict — the old code would have scored it as a "unanimous" 30-damage hit. The rejudge live test then closed the loop: a completely different panel changed 2 of 6 verdicts, flipped 67–83% of pairs individually… and produced the same ranking, Spearman 1.00. That is the pitch in one line: individual LLM judges are position-biased; the gated panel plus per-round BT is judge-robust, and the tool can prove it about its own output.

LIVE 03 · router gaps found, reported upstream

Two undocumented behaviors, one filed to the docs team

  • Anthropic reasoning tokens aren't counted. Claude with thinking enabled returns visible CoT + reasoning_signature, but completion_tokens_details.reasoning_tokens = 0 — while Gemini reports it correctly (e.g. 511). Feedback submitted via the docs channel.
  • Streamed CoT arrives as delta.reasoning (not the DeepSeek-style reasoning_content). The gateway now reads both — best-effort by contract, with the "thinking…" timer as the guaranteed path.

05The line-count truth

The plan projected ~1,600 src LOC for a like-for-like feature set. The repo landed at 2,426 src LOC (was 1,912) — and the delta deserves an honest itemization, because both plan numbers were achieved: everything slated for deletion died (judges/ 196, bracket + stub 100, audit dead code ~65 — zero hand-rolled judging or tournament code remains), and then ~870 lines of owner-approved new capability — none of which existed in the 1,912 — took its place:

  • rejudge.py + CLI command (~230) — the evaluatorq demo inside the demo;
  • statistics: bootstrap CIs, run manifest, final report, jury/win-grid leaderboard (~280);
  • reasoning support + timeout/void policy in gateway/battle/config (~180);
  • TUI upgrades: thinking states, tokens/sec, standings ticker, drama (~180).

Smaller where it duplicated, bigger where it differentiates. The complexity the audit condemned — speculative seams, reinvented libraries, dead surface — is gone; every added line traces to a plan decision with an owner sign-off. Tests grew 20 → 26 and now cover the semantics that carry money: void rounds, KO-continues-judging, draws, unanimity floors, and a headless render of the leaderboard.

Aggregate: A− (from B−). The B− codebase was a tidy toy wearing a distributed system's exoskeleton. What ships now is smaller where it was derivative, larger where it is differentiated, measured about its own uncertainty, and demonstrably judge-robust. The two products it exists to promote are no longer mentioned by the README — they are load-bearing: the router carries every token (and its registry audit reshaped the roster), and evaluatorq issues every verdict on screen.
